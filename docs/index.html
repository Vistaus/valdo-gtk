I'm missing a template. How to add it to the app?

Valdo GTK is based on Valdo, awesome terminal utility created by Princeton Ferro. So if you want to add a template to Valdo GTK, add it to Valdo and you will also see it in the next release of Valdo GTK.

So how to add a template? Just follow the instructions in the README file of the Valdo project.
